# Security Policy

We take the security of DocuMindAI seriously. If you believe you have found a security vulnerability, please report it to us privately following the guidelines below.

---

## Reporting a Vulnerability

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please send an email to:
👉 **security@documindai.com** (or contact the repository maintainer directly)

Please include the following details in your report:
- A description of the vulnerability and its potential impact.
- Step-by-step instructions (or a proof-of-concept script) to reproduce the issue.
- Details of the environment/system under which you tested it.

We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue promptly before public disclosure.

---

## Supported Versions

Only the latest version of the `main` branch of this repository is actively supported and receives security updates. If you are using a custom fork or an older release, we recommend pulling the latest changes from upstream.
